# 高频单词8297

一套面向中文英语学习者整理的高频词汇资料，共收录 **8297 条单词和短语**。内容以快速理解、结合语境记忆和长期复习为目标，适合导入 Anki，也可以直接在 GitHub 在线查阅。

## 你会学到什么

每条词汇尽量包含：

- **单词或短语**：涵盖常用词、词形变化和高频表达。
- **美式音标**：帮助确认读音和重音。
- **简明释义**：不同词性尽量分行，方便快速辨认。
- **英文例句**：在真实语境中理解词义和用法。
- **中文翻译**：用于核对例句理解。

Anki 原始卡牌还包含单词发音和例句发音。由于音频文件数量和体积较大，本仓库目前主要提供文本数据。

## 适合人群

- 想扩大日常英语词汇量的学习者
- 正在准备四六级、考研、雅思等考试的学习者
- 希望通过例句而不是孤立中文释义记单词的人
- 使用 Anki 进行间隔重复复习的人
- 需要 CSV 或 TSV 词汇数据进行二次整理的人

## 推荐学习方法

1. 看到正面后先读出单词，不要立刻翻面。
2. 尝试回忆最常用的词性和含义。
3. 翻面后朗读英文例句，结合上下文确认用法。
4. 暂时不要展开中文翻译，先自己理解句子。
5. 点击翻译核对，注意自己遗漏的搭配或语法。
6. 根据实际掌握程度选择 Anki 的复习评级。

每天新增少量新卡并坚持复习，通常比一次学习大量词汇更有效。

本项目用于个人英语学习和数据整理。使用、转载或再分发时，请同时遵守相关数据来源及第三方内容的使用条款。
